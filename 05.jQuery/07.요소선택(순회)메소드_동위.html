- %('선택자').siblings()

선택한 요소와 같은 레벨에 있는 모든 요소들 선택

- %('선택자').siblings('선택자')

선택한 요소와 같은 레벨에 있는 모든 요소들 중 제시한 값과 일치하는 요소들만 선택

- %('선택자').next|prev()

선택한 요소와 같은 레벨에 있는 바로 다음요소|이전요소 하나만 선택

- %('선택자').nextAll|prevAll()

선택한 요소와 같은 레벨에 있는 바로 다음요소들 모두|이전요소들 모두 하나만 선택

- %('선택자').nextUntil('선택자')

선택한 요소와 같은 레벨의 다음 요소들 중 제시한 값과 일치하는 요소 이전까지만 선택
