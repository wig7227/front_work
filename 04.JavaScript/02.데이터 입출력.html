자바스크립트에서의 데이터 입출력

* window : 자바스크립트 내장 객체로 브라우저 창이 열릴때마다 하나씩 만들어지는 객체
브라우저 창 안에 존재하는 모든 요소들의 최상위 객체(생략가능)

*document : 웹 문서마다 하나씩 만들어지는 개체(html문서에 대한 정보들을 가지고 있음)
