1. 속성 선택자

선택하고자 하는 요소내에 작성되어 있는 속성을 이용하여 선택하는 방법
선택자 뒤에 []를 이용하여 속성과 속성값을 제시함 

        선택자[속성명=속성값] { 스타일속성 : 값; ... } => 속성값이 "일치"하는 요소
        선택자[속성명~=속성값] { 스타일속성 : 값; ... } => 속성값이 단어로 "포함"되어 있는 요소 abc
        선택자[속성명^=속성값] { 스타일속성 : 값; ... } => 속성값으로 "시작"하는 요소
        선택자[속성명$=속성값] { 스타일속성 : 값; ... } => 속성값으로 "끝"나는 요소
        선택자[속성명*=속성값] { 스타일속성 : 값; ... } => 속성값으로 "포함"하는 요소

2. 자손 선택자 / 후손 선택자

요소들이 중첩되어 작성 가능 자손 : 바로 하위인 요소들 후손 : 하위요소들 전부 

        * 자손선택자 : >
        a>b {
            스타일속성:값;
        }

        * 후손선택자 : 공백
        a b {
            스타일속성:값;
        }

3. 동위 선택자

동위 관계(동등 관계)에서 바로 뒤에 위치한 특정 요소를 선택할 때 사용 

        *a요소 뒤에 나오는 b요소 하나만 선택 : +
        a+b {
            스타일속성:값;
        }

        *a요소 뒤에 있는 모든 b요소 선택 : ~
        a~b {
            스타일속성:값;
        }

4. 반응 선택자

사용자의 움직임에 따라 선택되는 선택자 

        * 해당 요소에 클릭이 되었을 경우 스타일 부여
        선택자:active{
            스타일속성:값;
        }

        * 해당 요소에 마우스가 올라가는 순간 스타일 부여
        선택자:hover{
            스타일속성:값;
        }

        * 링크 태그에서 방문하기 전의 스타일
        선택자:link{
            스타일속성:값;
        }

        * 링크 태그에서 방문한 후의 스타일
        선택자:visited{
            스타일속성:값;
        }

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자 

        1) 체크된(checked) 상태의 요소
           선택자:checked {
            스타일속성:값;
           }
    
        2) 초점(focus)가 맞춰진 input 요소
           선택자:focus {
            스타일속성:값;
           }

        3)활성화(enable)가 되어있는 요소
          선택자:enable {
          스타일속성:값;
        }
        4)비활성화(disable)가 되어있는 요소
          선택자:disable {
          스타일속성:값;
        }

6. 일반 구조 선택자

특정한 위치에 있는 태그를 선택하는 선택자(일반적으로 자손, 후손 선택자와 함께 사용)

        :first-child        -> 자식 관계 태그 중 첫번째 태그 선택
        :last-child         -> 자식 관계 태그 중 첫번째 태그 선택
        :nth-child(수열)    -> 자식 관계 태그 중 앞에서 수열 번째 태그 선택
        :nth-last-child(수열)   -> 자식 관계 태그 중 뒤에서 수열 번째 태그 선택
