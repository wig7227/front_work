flex

기본 축을 기준으로 정렬

        * 부모 요소 : flex-container
            - display : flex
            - flex-direction : row(기본값)|column|row-reverse|column-reverse
            - justify-content : 메인축정렬(flex-direction이 row라면 row, column이면 column이면 column)
            - align-items : 크로스축정렬(flex-direction이 row라면 column, column이면 row)
            - flex-wrap : 줄바꿈 여부
            - align-content : 여러줄일 경우 크로스축 설정
        * 자식 요소 : flex_item
            - flex-grow : 빈 여백을 아이템들의 주어진 비율대로 늘어나면서 레이아웃 영역 채움
            - flex-shrink : 넘치는 아이템 영역을 분배해서 레이아웃 영역안에 아이템들이 배치되도록 하는 방식
            - flex-basis : 아이템의 크기를 직접 설정
